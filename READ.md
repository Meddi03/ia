# Training Planner

**Автор:** Медведева Диана

## Описание

Training Planner — это приложение для планирования и отслеживания тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи по типу или дате, а также сохранять/загружать данные в формате JSON.

## Функциональность

- Добавление тренировок с валидацией ввода (формат даты, положительная длительность)
- Просмотр всех тренировок в таблице
- Фильтрация по типу тренировки (Бег, Велосипед, Плавание, Силовая, Йога)
- Фильтрация по конкретной дате
- Автоматическое сохранение в JSON-файл
- Загрузка данных при запуске приложения

## Требования

- Python 3.7+
- Стандартные библиотеки: tkinter, json, os, datetime

## Запуск

```bash
python training_planner.py
